# 🧪 Biomolecular Analysis with Python

  This is a Python-based molecular analysis project that demonstrates how to represent, visualize, and analyze biomolecules using **RDKit** and **Py3Dmol**. The notebook covers molecular structure visualization, molecular similarity calculations, descriptor analysis, atomic charge visualization, and interactive 3D molecular rendering.

## 📖 Overview

This project introduces key concepts in cheminformatics and molecular modeling, including:

- Molecular representation using **SMILES**
- 2D molecular visualization
- Atom indexing and highlighting
- Molecular descriptor calculation
- Molecular similarity analysis
- Atomic charge computation
- Interactive 3D molecular visualization
- Retrieval and visualization of real-world molecules and proteins

The notebook serves as a beginner-friendly introduction to computational chemistry and biomolecular analysis with Python.

---

## 🚀 Features

### 🔹 Molecular Representation
Convert molecular structures into RDKit molecule objects using SMILES notation.

### 🔹 2D Visualization
Generate molecular diagrams and highlight specific atoms or substructures.

### 🔹 Atom Analysis
- Display atom indices
- Count atoms
- Explicitly add hydrogen atoms

### 🔹 Molecular Similarity
Compare molecules using fingerprint-based similarity metrics such as:

- Tanimoto Similarity
- Dice Similarity

### 🔹 Molecular Descriptors
Calculate chemical properties including:

- Molecular Weight
- LogP
- Topological Polar Surface Area (TPSA)
- Hydrogen Bond Donors/Acceptors

### 🔹 Atomic Charge Visualization
Compute and visualize Gasteiger partial charges.

### 🔹 Interactive 3D Models
Generate and explore molecules in 3D using Py3Dmol.

### 🔹 Protein Visualization
Load and visualize protein structures from the Protein Data Bank (PDB).
